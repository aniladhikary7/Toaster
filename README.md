# Toaster
Sample Library

> Step 1. Add the JitPack repository to your build file

```gradle
allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
  ```
 > Step 2. Add the dependency
 
 ```gradle
 dependencies {
	        implementation 'com.github.aniladhikary7:Toaster:Tag'
	}
  ```
